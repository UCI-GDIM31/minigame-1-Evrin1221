Here's my analogy comparing components, game objects, and scenes to forming sentences. The components would be the letters, and putting them together would make words, which would be the game objects. Putting the words together in a specific order with spaces when necessary would form a coherent sentence, like how placing game objects in a specific way would make a scene.
